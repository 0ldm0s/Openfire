Fastpath Plugin Readme

Overview

The monitoring plugin adds chat archiving support to Openfire together with a neat dashboard and a list of reports.

Installation

Copy monitoring.jar into the plugins directory of your Openfire installation. The plugin will then be automatically deployed. To upgrade to a new version, copy the new monitoring.jar file over the existing file.

Make sure that the enterprise plugin is removed before installing this plugin. Otherwise the enterprise plugin will rename the fastpath.jar to fastpath.jar.old thus removing thus plugin. If this happens just stop the server, remove the enterprise.jar plugin and rename fastpath.jar.old to fastpath.jar and start the server again.

Configuration

Chat archiving is enabled by default. However, only meta data about conversations is enabled not include chat transcript archving. To enable chat transcript archiving or group chat archving you have to log into the admin console and go to Monitoring --> Archiving --> Archiving Settings..

Configuration

Run the corresponding migration script located in the database folder. You might need to start the server so that this plugin is exploded. You can also use an extracion tool like winzip to decompress this plugin and obtain the database scripts.
