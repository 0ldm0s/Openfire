Monitoring Plugin Readme

Overview

The monitoring plugin adds chat archiving support to Openfire together with a neat dashboard and a list of reports.

Installation

Copy monitoring.jar into the plugins directory of your Openfire installation. The plugin will then be automatically deployed. To upgrade to a new version, copy the new monitoring.jar file over the existing file.

Configuration

Chat archiving is enabled by default. However, only meta data about conversations is enabled not include chat transcript archving. To enable chat transcript archiving or group chat archving you have to log into the admin console and go to Monitoring --> Archiving --> Archiving Settings..
